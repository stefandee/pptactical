Engine features

  • Tanks (may fire while moving)
  • Artillery units (may only fire if standing)
  • Recons (tanks with high moving speed)
  • Transport (may only carry infantry units, may fire while moving)
  • Load action for Infantry units
  • Unload and Unload all actions for transports (unload is only available from transport info area)
  • Weapons explosions and projectiles for Spee and Lynx rendered
  • Ballistic projectile
  • Muzzle points
  • Path finding considers terrain type, propulsion type (hovers may pass over water) and altitude difference (slope for descending/ascending)
  • Units view now considers altitude (some sort of raycast)
  • Projectile trajectory takes into account altitudes
  • Unit info areas (brief info about units, group selections)
  • Smart screenshots
  • Mission scripting (see the PPT Script Language Reference.doc)
  • Added UnitType function for infantry scripting


Tools features

  • Added vehicle editing to UnitsEd (the gui was designed, but no editing was possible)
  • Corrected several issues in UnitsEd (MaskEdits increased capacity, added an edit box for infantry fire resistance)
  • Added turret turn rate parameter for vehicles (UnitsEd)
  • Added portrait for infantry and vehicles (UnitsEd)
  • Added turret turn rate, minimum range for vehicles weapons, projectiles/shot (P/shot), track persist time and after effect sprite for weapons (both infantry and vehicles), max slope for ascending and descending (both infantry and vehicles) and bay capacity for vehicles (UnitsEd)
  • Exit point for vehicles weapons in skeleton screen (UnitsEd)
  • Added option for importing a large sprite for tile; the sprite will be automatically be splitted into 96x96 cells - the corresponding bitmaps and .dats will be generated (MapChunkEd)
  • Re-added the altitude editor (MapEditor)
  • Added vehicle editing in unit tree (MapEditor)
